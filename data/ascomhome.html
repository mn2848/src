myFP2ESP32 ASCOM ALPACA SERVER

This is an ASCOM ALPACA SERVER that provides focuser control for an ASCOM Remote client.

The ASCOM Alpaca Server is running on a myFP2ESP32 WiFi controller.

  • full remote control
  • any ASCOM remote client can connect to this Server

    • If you do not have an ASCOM Remote client, you can use the free one provided with the ASCOM Platform. Alternatively, the myFP2ESP32 project provides an ASCOM Driver which can be used to access the myFP2ESP controller via TCP/IP.
